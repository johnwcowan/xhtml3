                            34. XML Scripting Module

   This section is normative for purposes of defining the integration of
   the XML Script Module into XHTML 2. The semantics of XML Scripting
   Module itself are normatively defined in [[10]XMLEVENTS]. As permitted
   by [[11]XMLEVENTS], this element is incorporated into the XHTML
   namespace.

   This module defines the script element as follows:

   Element Attributes Content Model
   script [12]Common,
   charset ([13]Charset),
   defer ("defer"),
   implements ([14]URIorSafeCURIEs) PCDATA

   Implementations: [15]RelaxNG, [16]XML Schema
