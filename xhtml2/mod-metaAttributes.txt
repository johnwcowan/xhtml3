                  24. XHTML Metainformation Attributes Module

   Contents
     * 24.1. [10]Metadata Attribute Collection

   This section is normative for purposes of defining the integration of
   the XHTML Metainformation Attributes Module into XHTML 2. The semantics
   of the XHTML Metainformation Attributes Module itself are normatively
   defined in [[11]RDFASYNTAX]. The rules for extracting RDF from XHTML
   family markup languages are defined in [[12]RDFASYNTAX]. For
   information on important differences between XHTML 2 and other XHTML
   family markup languages and how those may relate to RDFa, see
   [13]Appendix A.

24.1. Metadata Attribute Collection

   Attributes Notes
   about ([14]URIorSafeCURIE)
   content ([15]CDATA)
   datatype ([16]CURIE) If not specified, then the default value is string
   as defined in [[17]XMLSCHEMA].
   typeof ([18]CURIEs)
   property ([19]CURIEs)
   rel (reserved word | [20]CURIE)+ See the reserved values list in
   [[21]XHTMLVOCAB]
   resource ([22]URIorSafeCURIE)
   rev (reserved word | [23]CURIE)+ See the reserved values list in
   [[24]XHTMLVOCAB]

   Implementations: [25]RelaxNG, [26]XML Schema
