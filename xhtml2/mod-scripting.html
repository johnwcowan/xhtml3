34. XML Scripting Module

This section is normative for purposes of defining the integration of the XML Script Module into XHTML 2. The semantics of XML Scripting Module itself are normatively defined in [XMLEVENTS]. As permitted by [XMLEVENTS], this element is incorporated into the XHTML namespace.

This module defines the script element as follows:

Element Attributes Content Model
script Common,
charset (Charset),
defer ("defer"),
implements (URIorSafeCURIEs) 		PCDATA

Implementations: RelaxNG, XML Schema
