26. XHTML Role Attribute Module

Contents

  • 26.1. Role Attribute Collection

This section is normative for purposes of defining the integration of the XHTML Role Module into XHTML 2. The semantics of the XHTML Role Module itself are normatively defined in [XHTMLROLE].

This module defines the role attribute and the associated Role Attribute Collection. The Role Attribute Collection is included in the Common attribute collection.

26.1. Role Attribute Collection

Attribute Notes
role (CURIEs)  

The XHTML Role Attribute allows the author to annotate XHTML Documents with machine-extractable semantic information about the purpose of an element. Use cases include accessibility, device adaptation, server-side processing, and complex data description.

Implementations: RelaxNG XML Schema
