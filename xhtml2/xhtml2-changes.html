A. Changes from earlier XHTML versions

This appendix is informative.

This Appendix describes the differences between XHTML 2.0 and XHTML 1.1. It also describes some important ways that changes in XHTML 2.0 may effect creators of RDFa processors [RDFASYNTAX].
