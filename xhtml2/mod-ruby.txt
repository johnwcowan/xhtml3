                                27. Ruby Module

   This section is normative for purposes of defining the integration of
   the Ruby Module into XHTML 2. The semantics of the Ruby Module itself
   are normatively defined in [[10]RUBY].

   This module adds the ruby element to the [11]Text content set of the
   [12]Text Module. XHTML 2.0 supports the maximal content model for the
   ruby element, defined as follows:
((rb, (rt | (rp, rt, rp))) | (rbc, rtc, rtc?))

   As defined in [[13]RUBY], the ruby element is not allowed to nest.

   Implementations: [14]RELAX NG, [15]XML Schema, [16]DTD
